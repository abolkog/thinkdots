## Run it locally

1- Clone the Repo

2- If you are using `nvm`, setup node version using the `.nvmrc` file

```shell
nvm use
```

3- Install the dependencies

```shell
npm i
```

4- Start the dev server

```shell
npm run dev
```

5- Visit [http://localhost:3000](http://localhost:3000) on your browser.
